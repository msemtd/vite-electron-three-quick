# vite-electron-three-quick

> vite + electron + three.js

The very simplest quick start for vite and electron with three.js (no Vue, no TypeScript!)

## Install

```
$ yarn
```

## Development server with Vite

```
$ yarn run dev
```

## Build an installer

```
$ yarn run build
```
